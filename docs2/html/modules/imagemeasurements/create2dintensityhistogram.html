Create 2D intensity histogram

Creates a 2D intensity histogram for a pair of specified images.

Description

Creates a 2D intensity histogram for a pair of specified images. Intensities along the x-axis correspond to the first input image and those along the y-axis to the second input image. Output histogram is saved to the workspace as an image. Works for N-dimensional image stacks (must have the same dimensions). Uses the ImgLib2 implementation ND intensity histograms.

Parameters

Input image 1 (x-axis) (default = "") First image from the workspace used in 2D intensity histogram plotting. Intensities from this image are represented along te output histogram x-axis (columns). For multidimensional stacks, all pixel values are compiled into a single intensity histogram.

Input image 2 (y-axis) (default = "") Second image from the workspace used in 2D intensity histogram plotting. Intensities from this image are represented along te output histogram y-axis (rows). For multidimensional stacks, all pixel values are compiled into a single intensity histogram.

Output image (default = "") Output 2D intensity histogram, which will be saved to the workspace.

Minimum bin (image 1) (default = "0.0") Minimum intensity bin for the image specified by "Input image 1 (x-axis)".

Maximum bin (image 1) (default = "255.0") Maximum intensity bin for the image specified by "Input image 1 (x-axis)".

Number of bins (image 1) (default = "256") Number of intensity bins to use for the image specified by "Input image 1 (x-axis)".

Include tail bin (image 1) (default = "false") When selected, additional bins at the extremes of the histogram x-axis will be used for intensity values that fall outside the specified intensity range.

Minimum bin (image 2) (default = "0.0") Minimum intensity bin for the image specified by "Input image 2 (y-axis)".

Maximum bin (image 2) (default = "255.0") Maximum intensity bin for the image specified by "Input image 2 (y-axis)".

Number of bins (image 2) (default = "256") Number of intensity bins to use for the image specified by "Input image 2 (y-axis)".

Include tail bin (image 2) (default = "false") When selected, additional bins at the extremes of the histogram y-axis will be used for intensity values that fall outside the specified intensity range.
