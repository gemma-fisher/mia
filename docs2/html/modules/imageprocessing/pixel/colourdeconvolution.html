Colour deconvolution

Applies the

Description

Applies the Colour Deconvolution plugin to unmix an RGB image (stored in the workspace) into up to three separate channels, which are output as separate images. The input image can be stored as either an RGB or composite image. This process is only applicable to images created via subtractive mixing (e.g. histological staining), not to additive methods (e.g. fluorescence).

Parameters

Input image: Image from the workspace to apply unmixing to. This can be stored as either an RGB or composite image.

Output image 1: When selected, the first stain in the stain matrix will be output to the workspace with the name specified by "Output image 1 name"

Output image 1 name: Name to assign to first stain image, if output to the workspace.

Output image 2: When selected, the second stain in the stain matrix will be output to the workspace with the name specified by "Output image 2 name"

Output image 2 name: Name to assign to second stain image, if output to the workspace.

Output image 3: When selected, the third stain in the stain matrix will be output to the workspace with the name specified by "Output image 3 name"

Output image 3 name: Name to assign to third stain image, if output to the workspace.

Stain model: Stain models to apply to input image. If set to "Custom (user values)" the individual RGB components for each channel can be specified. Model choices are: Alcian blue & H,Azan-Mallory,Brilliant_Blue,CMY,FastRed FastBlue DAB,Feulgen Light Green,Giemsa,H AEC,H&E,H&E 2,H&E DAB,H DAB,H PAS,Masson Trichrome,Methyl Green DAB,RGB,Custom (user values)

Stain 1 (red): Red component of stain 1. Value specified in range 0-1.

Stain 1 (green): Green component of stain 1. Value specified in range 0-1.

Stain 1 (blue): Blue component of stain 1. Value specified in range 0-1.

Stain 2 (red): Red component of stain 2. Value specified in range 0-1.

Stain 2 (green): Green component of stain 2. Value specified in range 0-1.

Stain 2 (blue): Blue component of stain 2. Value specified in range 0-1.

Stain 3 (red): Red component of stain 3. Value specified in range 0-1.

Stain 3 (green): Green component of stain 3. Value specified in range 0-1.

Stain 3 (blue): Blue component of stain 3. Value specified in range 0-1.
