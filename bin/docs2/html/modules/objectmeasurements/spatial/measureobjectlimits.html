Measure object limits

Measures the XYZ spatial limits of each object relative to the origin (x = 0, y = 0, z = 0) of the original image.

Description

Measures the XYZ spatial limits of each object relative to the origin (x = 0, y = 0, z = 0) of the original image. Limits are stored as measurements associated with each object. Measurements are reported in pixel (slice for z) coordinates and calibrated units.

Parameters

Input objects (default = "Objects from workspace to measure centroid of. Measurements will be associated with the corresponding object in this collection.") Objects to measure spatial limits for. Measurements will be associated with each object.
