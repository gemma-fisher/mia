Image measurement calculator

Perform a mathematical operation on measurements associated with an image.

Description

Perform a mathematical operation on measurements associated with an image. The calculation can replace either or both input image measurements with fixed values. The resulting measurement is associated with the input image as a new measurement.

Parameters

Input image (default = "") Image from the workspace to perform the measurement calculation for.

Value mode 1 (default = "Measurement") Controls how the first value in the calculation is defined:
  • "Fixed" A single, fixed value defined by "Fixed value 1"is used.
  • "Measurement" A measurement associated with the input image and defined by "Measurement 1" is used.

Fixed value 1 (default = "0.0") Fixed value to use in the calculation when "Value mode 1" is in "Fixed" mode.

Measurement 1 (default = "") Measurement associated with the input image to use in the calculation when "Value mode 1" is in "Measurement" mode.

Value mode 2 (default = "Measurement") Controls how the second value in the calculation is defined:
  • "Fixed" A single, fixed value defined by "Fixed value 2"is used.
  • "Measurement" A measurement associated with the input image and defined by "Measurement 2" is used.

Fixed value 2 (default = "0.0") Fixed value to use in the calculation when "Value mode 2" is in "Fixed" mode.

Measurement 2 (default = "") Measurement associated with the input image to use in the calculation when "Value mode 2" is in "Measurement" mode.

Output measurement (default = "") The value resulting from the calculation will be stored as a new measurement with this name. This output measurement will be associated with the input image

Calculation mode (default = "Add value 1 and value 2") Calculation to perform. Choices are: Add value 1 and value 2, Divide value 1 by value 2, Multiply value 1 and value 2, Subtract value 2 from value 1.
