Add custom metadata item

Description

DEPRECATED: Please use "Global variables" module instead, which allows custom values to be accessed using the V{[NAME]} notation. Global variables can also be stored as metadata items for the purpose of exporting.

This module allows for a specific metadata item to be used. An example of this would be to add a label for generic (metadata-based) filename generation in the image loader (i.e. all images to be loaded must have the word "phase" in them).

Parameters

  • Metadata name (default = "") Name for metadata item to be assigned.

  • Metadata value (default = "") Value of metadata item. This will be the same for all analysis runs performed at the same time since this value doesn't update during an analysis.
