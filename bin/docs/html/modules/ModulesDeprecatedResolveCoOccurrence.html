Resolve co-occurrence

Description

Identifies overlapping objects and moves them to a new object collection Objects are linked on a one-to-one basis.

Parameters

  • Input objects 1 (default = "")

  • Input objects 2 (default = "")

  • Output objects name (default = "")

  • Overlap mode (default = "Spatial overlap")

  • Maximum separation (default = "1.0")

  • Calibrated units (default = "false") When selected, spatial values are assumed to be specified in calibrated units (as defined by the "Input control" parameter "Spatial unit"). Otherwise, pixel units are assumed.

  • Minimum overlap of object 1 (%) (default = "50.0")

  • Minimum overlap of object 2 (%) (default = "50.0")
