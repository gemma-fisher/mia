Measure relative orientation

Description

Currently only works for X-Y plane measurements

Parameters

  • Input objects (default = "")

  • Orientation mode (default = "Orientation in X-Y plane")

  • Orientation in X/Y measurement (default = "")

  • Orientation in XY/Z measurement (default = "")

  • Measurement range (default = "0-90 degrees")

  • Reference mode (default = "Object centroid to image centre")

  • Reference objects (default = "")

  • Object choice mode (default = "Largest object")

  • Reference must be in same frame (default = "true")
