Project image

Description

Project an image along the z-axis into the xy-plane. Various statistics are available at each pixel location (e.g. maximum, mean, etc.).

Parameters

  • Input image (default = "") Image to be projected.

  • Output image (default = "") Name for the projected image to be stored in the workspace with.

  • Projection mode (default = "Maximum") Statistic for calculation of the intensity projection.
