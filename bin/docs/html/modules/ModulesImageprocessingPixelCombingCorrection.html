Combing correction

Description

Applies an integer pixel row shift to every other row (starting with top-most row).

Parameters

  • Input image (default = "") Image to which the correction will be applied.

  • Apply to input image (default = "true") Set to "true" to apply correction to the input image or "false" to store the corrected image separately.

  • Output image (default = "") Output image with correction applied.

  • Offset (px) (default = "0") Pixel offset to be applied to every other row.
