Fit Gaussian 2D

Description

Gaussian spot fitting. Can take objects as estimated locations. ***Only works in 2D*** ***Only works for refinement of existing spots***

Parameters

  • Input image (default = "")

  • Input objects (default = "")

  • Method to estimate spot radius (default = "Fixed value")

  • Radius (default = "1.0")

  • Radius measurement (default = "")

  • Measurement multiplier (default = "1.0")

  • Limit sigma range (default = "true")

  • Minimum sigma (x Radius) (default = "0.25")

  • Maximum sigma (x Radius) (default = "4.0")

  • Fixed fitting window (default = "false")

  • Window size (default = "15")

  • Maximum number of evaluations (default = "1000")

  • Remove objects with failed fitting (default = "false")

  • Apply volume (default = "true")

  • Create Gaussian image (default = "false")

  • Gaussian image name (default = "")
