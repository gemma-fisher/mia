Focus stack

Description

Channels will be focused individually. Uses the StackFocuser plugin created by Mikhail Umorin. Source downloaded from https://imagej.nih.gov/ij/plugins/download/Stack_Focuser_.java on 06-June-2018

Parameters

  • Input image (default = "")

  • Output focused image (default = "")

  • Use existing height image (default = "false")

  • Input height image (default = "")

  • Range (default = "11")

  • Smooth height map (default = "true")

  • Add height map image to workspace (default = "false")

  • Output height image (default = "")

  • Show height image (default = "false")
