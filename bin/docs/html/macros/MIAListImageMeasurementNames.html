MIA_ListImageMeasurementNames

Description

Returns a list of available measurements for the specified image. Use the macro Ext.MIA_ShowAllImageMeasurements(imageName) to get a table containing all measurement values for the image. Use the macro Ext.MIA_GetImageMeasurement(imageName, measurementID) to get a specific measurement value for a specific image.

Parameters

  • String imageName
