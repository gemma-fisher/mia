Create distance map

Description

Parameters

  • Input objects (default = "")

  • Output image (default = "")

  • Reference mode (default = "Distance from object centroid")

  • Invert map within objects (default = "true")

  • Masking mode (default = "Inside and outside")

  • Normalise map per object (default = "false")

  • Spatial units mode (default = "Pixel/slice") Controls whether spatial values are assumed to be specified in calibrated units (as defined by the "Input control" parameter "Spatial unit") or pixel units.
