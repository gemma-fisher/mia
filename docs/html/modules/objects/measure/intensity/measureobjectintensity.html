Modules > Objects > Measure > Intensity

Measure object intensity

Measure intensity of each object in a specified image.

Description

Measure intensity of each object in a specified image. Measurements of intensity are taken at all pixel coordinates corresponding to each object. By default, basic measurements such as mean, minimum and maximum will be calculated. Additional measurements can optionally be enabled.

Parameters

Parameter Description
Input objectsObjects from the workspace for which intensities will be measured.
Input imageImage from which pixel intensities will be measured. This image can be 8-bit, 16-bit or 32-bit. Measurements are always taken from the first channel if more than one channel is present (to measure additional channels, please first use the "Extract substack" module).
Measure weighted centreWhen selected, the intensity-weighted centroid of each input object will be calculated. With this, the greater the intensity in a particular region of an object, the more the "centre of mass" will be drawn towards it.
