Measure image colocalisation

Description

Calculates colocalisation of two input images. All measurements are associated with the first input image. Measurements can be restricted to specific region using image or object-based masking. To measure colocalisation on an object-by-object basis please use the "Measure object colocalisation" module.

All calculations are performed using the Coloc2 plugin.

Parameters

  • Input image 1 (default = "")

  • Input image 2 (default = "")

  • Masking mode (default = "None")

  • Mask image (default = "")

  • Image mask logic (default = "Measure on white (255 intensity)")

  • Input objects (default = "")

  • Object mask logic (default = "Measure inside objects")

  • Thresholding mode (default = "Bisection (correlation)")

  • Threshold (C1) (default = "1.0")

  • Threshold (C2) (default = "1.0")

  • PCC implementation (default = "Fast")

  • Measure Kendall's Rank Correlation (default = "true")

  • Measure Li's ICQ (default = "true")

  • Measure Manders' Correlation (default = "true")

  • Measure PCC (default = "true")

  • Measure Spearman's Rank Correlation (default = "true")
