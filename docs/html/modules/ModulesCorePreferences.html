Preferences

Description

General MIA parameters.

Parameters

  • Show deprecated modules (editing mode) (default = "false") When selected, deprecated modules will appear in the editing view available modules list. These modules will be marked with a strikethrough their name, but otherwise act as normal. Note: Modules marked as deprecated will be removed from future versions of MIA.
