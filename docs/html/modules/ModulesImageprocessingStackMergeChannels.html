Merge channels

Description

DEPRECATED: This Module has been superseeded by the more generalised "Concatenate stacks" Module. It will be removed in a future release.

Combines image stacks as different channels. Output is automatically converted to a composite image.

Parameters

  • Add image (default = "") Add another image to be included in output merged image. All added images must have the same X,Y,Z and T dimensions.

  • Input image (default = "") Image from workspace to add to output merged image.

  • Overwrite mode (default = "Create new image") Controls where the output image is stored:
    • "Create new image" Stores the merged image as a new image in the workspace with the name specified by "Output image".
    • "Overwrite image" Overwrite the image specified by the index "Image index to overwrite (>= 1)" in the workspace with the merged image.

    • Output image (default = "") Name for the output merged image to be stored in the workspace with.

    • Image index to overwrite (>= 1) (default = "1") If overwriting one of the input images, the image specified by this index (numbering starting at 1) will be overwritten.
